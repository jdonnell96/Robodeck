Why RigStack

One-Click Install

Install tools like Label Studio, PyTorch, ROS 2, and more with a single click.

Live Monitoring

Health checks, status LEDs, and streaming install logs in real time.

Secure by Design

Command allowlist, shell injection blocking, and user consent before every action.

Cross-Platform

Native app for macOS, Windows, and Linux at just ~15MB.

Curated Catalog

Every tool vetted by the RigStack team. No unreviewed community packages.

Managed Environments

Isolated installs in ~/.rigstack/ — never touches your system Python or npm.
